## Getting Started

1. **Clone the repository:**

    ```bash
    git clone https://github.com/sachinaryan912/buybazaar.git
    ```

2. **Navigate to the project directory:**

    ```bash
    cd buybazaar-spring
    ```

3. **Build the project:**

    ```bash
    mvn clean install
    ```

4. **Run the application:**

    ```bash
    mvn spring-boot:run
    ```
